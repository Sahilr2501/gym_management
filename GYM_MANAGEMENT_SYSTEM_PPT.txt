SLIDE 9: TECHNOLOGY STACK
-------------------------
Frontend Framework:
• React 18.2.0 - Modern UI library with function components and hooks

Routing:
• react-router-dom 6.20.0 - Client-side routing

State Management:
• Context API - Built-in React state management
• useReducer - Advanced state management

Build Tools:
• Vite 5.0.8 - Fast build tooling and development server

Styling:
• Tailwind CSS 3.3.6 - Utility-first CSS framework

Form Handling:
• react-hook-form 7.48.2 - Performant form library

Utilities:
• date-fns 2.30.0 - Date formatting and manipulation

SLIDE 10: PROJECT ARCHITECTURE
------------------------------
Application Structure:

src/
├── components/
│   ├── layout/          (Header, Sidebar, Footer, Layout)
│   ├── members/         (MemberCard, MembersList, MemberForm, MemberDetails)
│   ├── memberships/     (PlansList, PlanForm, PlanSelect)
│   ├── attendance/      (AttendanceLog)
│   ├── payments/        (PaymentsList, PaymentForm)
│   └── dashboard/       (StatCard, RecentActivity)

├── contexts/            (MembersContext, PlansContext)
├── data/               (mockData.js - initial data)
├── pages/              (Dashboard, MembersPage, PlansPage, ReportsPage, etc.)
├── services/           (localStorage helpers)
└── styles/             (Tailwind CSS configuration)

SLIDE 11: ROUTING STRUCTURE
---------------------------

  - Complete member profile view

• /members/:id/edit (Edit Member)
  - Member information update

• /plans (Membership Plans)
  - All plans listing

• /plans/new (Add New Plan)
  - Plan creation form

• /plans/:id/edit (Edit Plan)
  - Plan modification

• /reports (Reports & Statistics)
  - Comprehensive analytics

SLIDE 12: STATE MANAGEMENT
--------------------------
Context API Implementation:

MembersContext:
• Manages all member data
• Handles CRUD operations (Create, Read, Update, Delete)
• Manages attendance tracking
• Handles payment records
• Automatic localStorage persistence

PlansContext:
• Manages all membership plans
• Handles plan CRUD operations
• Links plans to members
• Automatic localStorage persistence

Benefits:
• Centralized state management
• Easy data sharing across components
• Automatic data persistence
• No external state management library needed

==========================================

SLIDE 20: INSTALLATION & SETUP
------------------------------
Prerequisites:
• Node.js (LTS version recommended)
• npm or yarn package manager

Installation Steps:
1. Clone the repository
2. Navigate to project directory
3. Run: npm install
4. Start dev server: npm run dev
5. Open browser at http://localhost:3000

Build Commands:
• npm run dev - Development server
• npm run build - Production build
• npm run preview - Preview production build
